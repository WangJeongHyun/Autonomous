## 프로젝트 소개

신호등 인식을 통한 자율주행 차량 구동 

- **프로젝트 기간** : 23.11.11 ~ 23.12.5
- **프로젝트 인원**
  - PM : 김규원
  - 신호등 영상처리 : 왕정현, 김지민
  - ROS 구동 및 연동 : 이현우, 정주환 
- **프로젝트 기능**
  - 신호등 처리를 통한 빨간불, 파란불 인식
  - 차선인식, 정지선 검출
  - 빨간불일 때, 정지선 앞에서 차량 멈춤
  - 초록불 판단 후 차량 출발
